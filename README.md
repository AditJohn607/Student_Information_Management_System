# 🎓 Student Information Management System

A complete **ASP.NET MVC + Entity Framework** based web application designed to manage all core academic operations of an educational institution including students, staff, attendance, exams, library, notices, timetable, assignments, and fee management.

---

## 📌 Project Overview

The **Student Information Management System (SIMS)** is a centralized platform that helps institutions digitally manage student-related data and academic activities efficiently. It reduces manual work and improves data accuracy, accessibility, and reporting.

---

## 🚀 Key Features

- Student Registration & Management
- Parent Information Module
- Staff Management System
- Attendance Tracking System
- Fee Management System with Payment Tracking
- Exam & Result Management
- Library Issue & Return System
- Notice Board System
- Timetable Management
- Assignment Management System
- Dashboard with Analytics
- Receipt Generation for Fee Payments

---

## 🧩 Modules Breakdown

### 1. Student Module
- Add, update, delete student records
- View student profiles

### 2. Parent Module
- Store parent/guardian details linked with students

### 3. Staff Module
- Manage teaching and non-teaching staff

### 4. Attendance Module
- Daily attendance tracking
- Attendance reports

### 5. Fee Module 💰
- Fee structure creation
- Payment tracking
- Balance calculation
- Fee receipt generation
- Dashboard analytics (monthly collection, payment mode distribution)

### 6. Exam Module
- Exam scheduling
- Marks entry and result generation

### 7. Library Module
- Book issue and return system
- Fine calculation (if applicable)

### 8. Notice Module
- Admin notice posting system
- Student notifications

### 9. Timetable Module
- Class schedule management

### 10. Assignment Module
- Assignment upload and submission tracking

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript, React, Bootstrap
- **Backend:** ASP.NET MVC (C#)
- **ORM:** Entity Framework
- **Database:** SQL Server
- **IDE:** Visual Studio 2013+

---

## 🗄️ Database Design

The system uses relational database design with tables such as:

- Students
- Parents
- Staff
- Attendance
- FeeStructure
- FeePayment
- Exams
- Library
- Notices
- Timetable
- Assignments
